Sobre imagens

para criar links com imagens ou videos ou palavras é só enveolapar a palavra com a tag "a" e colocar o referencial ou o link que voce queira

para adicionar uma imagem escrever img + shearch e aperta crtl+espaço para preocurar uma imgaem(obs:a imagem tem que estar dentro da pasta de arquivos)

imagens tem que ser pequenas mas não com má qualidade

para escrever codigos usa tag code antes de escrever os codigos

por exemplo imagine que isso é um codiog asb278237 12631827 gha8st81 2

link internos são as suas propias poginas por exemplo voce quer passar para a segunda a pagina então voce coloca o nome da sua pagina ex: pag3.html ou usa crtl+espaço para preocura ela e colocar no "rel" a tag self para indicar que voce esta indo para a sua propia pagina

para link externos usar a tag "target" e "_blank" dentro do link do site que voce queira que vá para que abra outra pagina quando clicar no link e a sua pagina não precisa fechar para voce ir e usar no "rel" a tag external para indicar que é externo que você está indo para fora do site
